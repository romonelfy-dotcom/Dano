# Dano - Countryballs Game 🌍

¡Bienvenido al juego de **Countryballs**! Un juego interactivo basado en web donde controlas países personificados como esferas.

## 🎮 Jugar Ahora

### Opción 1: Directo en GitHub Pages
**[JUGAR AQUÍ](https://romonelfy-dotcom.github.io/Dano/)**

### Opción 2: Ver en el navegador (sin descargar)
Abre el archivo `index.html` directamente:
- Ve a: https://github.com/romonelfy-dotcom/Dano/blob/main/index.html
- Haz clic en el botón **"Raw"** para verlo en el navegador
- O usa un visor online: https://htmlpreview.github.io/?https://github.com/romonelfy-dotcom/Dano/blob/main/index.html

## 📋 Controles

- **Enter**: Iniciar juego
- **Flechas/WASD**: Mover tu countryball
- **Mouse/Click**: Atacar enemigos

## 🎯 Objetivo

Defiende tu país (countryball) eliminando enemigos y ganando puntos. ¡Sube tu puntuación lo máximo posible!
